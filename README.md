Customer Churn Analysis | Power BI Internship
Organization: @Codveda

Intern: Maleesha Rasanjali

Domain: Power BI & Data Analytics

🎯 Project Objective
This project involves transforming raw customer data into actionable business intelligence to identify churn patterns and regional risks.

🛠️ Technical Implementation
Level 1: Data Preparation (ETL)
Imported and cleaned datasets using Power Query.

Handled null values and optimized data types for consistency.

Level 2: Data Modeling & DAX
Implemented a Star Schema by creating a dedicated 'States' dimension table.

Developed foundational measures: Total Customers, Churned Customers, and Churn Rate %.

Level 3: Advanced Analytics
Advanced DAX: Utilized RANKX for regional prioritization and SWITCH for dynamic risk categorization.

Security: Implemented Row-Level Security (RLS) to demonstrate enterprise-grade data governance.
